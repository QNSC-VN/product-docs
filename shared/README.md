# Shared assets

Cross-project resources reused across QNSC-VN product designs.

Suggested contents:
- **brand/** — logos, color palette, typography
- **design-system/** — shared component references, Figma links
- **references/** — style guides, external standards, links to dashboards/tickets

Keep large editable binaries (PSD, high-res video) in Figma/Drive and link to them here rather than committing them.
